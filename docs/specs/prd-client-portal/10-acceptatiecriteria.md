- [ ] Stefan Roevros kan inloggen via magic link en ziet uitsluitend CAI-issues

- [ ] Tab-filter Bugs/Features werkt
- [ ] Issue-detailpagina toont `client_title`/`client_description` met fallback
- [ ] Productie-issues worden automatisch zichtbaar via label-regel
- [ ] JAIP-admin kan in Cockpit handmatig `client_visible` toggelen
- [ ] RLS getest: testaccount van andere klant ziet geen CAI-data

- [ ] Audit-log registreert visibility-wijzigingen
- [ ] Stefan heeft minimaal één keer ingelogd en het overzicht bekeken (validatie-meeting gepland)
