# Golden Questions — Evaluatie-set Fase 1

Handmatig doorlopen na elke fase om regressie te detecteren.
Verwachte antwoorden invullen na eerste echte meeting-verwerking.

## Vragen

| #   | Vraag                                                             | Verwacht antwoord                           | Bron (meeting) | Status |
| --- | ----------------------------------------------------------------- | ------------------------------------------- | -------------- | ------ |
| 1   | "Wat is er besloten in de meeting van [datum]?"                   | Specifiek besluit met transcript_ref        |                | [ ]    |
| 2   | "Welke actiepunten staan open voor [naam]?"                       | Lijst met assignee, deadline, project       |                | [ ]    |
| 3   | "Wat wil [klant] van ons?"                                        | Needs met urgency en client context         |                | [ ]    |
| 4   | "Wat zijn de belangrijkste inzichten uit de laatste client call?" | Insights met category "client_feedback"     |                | [ ]    |
| 5   | "Welke projecten zijn besproken in de afgelopen week?"            | Lijst projecten met meeting-links           |                | [ ]    |
| 6   | "Zijn er tegenstrijdige besluiten genomen?"                       | Conflicterende decisions of "nee"           |                | [ ]    |
| 7   | "Wat is de status van [project]?"                                 | Project updates uit recente meetings        |                | [ ]    |
| 8   | "Welke risico's zijn gesignaleerd?"                               | Insights met category "risk" of "strategic" |                | [ ]    |
| 9   | "Wat heeft [persoon] afgelopen sprint gedaan?"                    | Action items + decisions van die persoon    |                | [ ]    |
| 10  | "Geef een samenvatting van alle meetings met [organisatie]"       | Meetings gefilterd op organization_id       |                | [ ]    |

## Instructies

1. Verwerk een echte Fireflies webhook
2. Vul de verwachte antwoorden in op basis van de meeting-inhoud
3. Stel elke vraag via Claude/MCP
4. Markeer met [x] als het antwoord correct en volledig is
5. Noteer afwijkingen in een kolom "Opmerkingen"

## Wanneer doorlopen

- Na sprint 5 (pipeline compleet)
- Na elke volgende sprint die de pipeline of search raakt
- Bij model-upgrades of prompt-wijzigingen
